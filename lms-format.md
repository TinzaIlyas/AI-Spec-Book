### Content Standards for LMS Compatibility

#### File Naming Convention
```
Format: [Type]_[Week]_[Topic]_[Version].[extension]
Examples:
- LEC_01_Introduction_v1.0.pdf
- VID_03_ForwardKinematics_v1.0.mp4
- LAB_05_PIDController_v1.0.py
- ASG_07_RLImplementation_v1.0.pdf
```

#### Video Standards
- **Resolution**: Minimum 720p, recommended 1080p
- **Format**: MP4 (H.264 codec)
- **Length**: 10-15 minutes per video for optimal engagement
- **Captions**: SRT format for accessibility
- **Audio**: Clear, noise-free, 48kHz minimum

#### Document Standards
- **Format**: PDF for consistency across platforms
- **Font**: Sans-serif for readability (Arial, Helvetica, Calibri)
- **Size**: 11-12pt for body text, 14-16pt for headings
- **Margins**: 1-inch standard
- **Accessibility**: Tagged PDFs with proper heading structure
